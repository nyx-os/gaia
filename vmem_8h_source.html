123 /*
124 Allocates size bytes at offset phase from an align boundary such that the resulting segment
125 [addr, addr + size) is a subset of [minaddr, maxaddr) that does not straddle a nocross−
126 aligned boundary. vmflag is as above. One performance caveat: if either minaddr or maxaddr is
